# Frontend Development Test - Restaurant App

## 💻 Tech Stack
- **React.js Version**:  (menggunakan Vite)
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **Routing**: React Router DOM

## 🚀 How to Start the Project
1. Clone repository ini ke komputer Anda.
2. Buka terminal dan arahkan ke folder project.
3. Jalankan perintah `npm install` untuk mengunduh semua dependency.
4. Buat file `.env` di root folder dan tambahkan variabel berikut:
   `VITE_API_URL=https://[ID-MOCKAPI-ANDA].mockapi.io/restaurants`
5. Jalankan perintah `npm run dev` untuk menyalakan local server.
6. Buka `http://localhost:5173` di browser.
